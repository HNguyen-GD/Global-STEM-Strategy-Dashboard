# Global STEM Strategy Dashboard
## Overview
Engineered a strategic Tableau dashboard by integrating large-scale U.S. education datasets (Scorecard & Field of Study) to track institutional ROI and enrollment trends.

## Tools & Techniques
- **Tools:** Tableau, Excel
- **Techniques:** Data Integration, Strategic Visualization, Market Analysis

## Key Findings
- Identified high-ROI disciplines and underutilized geographic markets.
- Delivered actionable growth insights for Xavier University, enabling data-driven expansion strategies.

## Repository Contents
- `/dashboard`: Tableau workbook/screenshot of the final dashboard.
- `/data_processing`: Excel files used for data cleaning and preparation.
